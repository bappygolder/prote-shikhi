# CTX-01 — Bootstrap First Flashcard MVP

**Status**: Done
**Project**: Bornomala
**Last updated**: 2026-05-04

## Objective

Scaffold the Expo React Native app and build the first mobile-first human-assisted flashcard trainer.

## Scope

- Expo React Native + TypeScript setup
- Bangla vowel cards
- Bangla-only learner card face
- Right/Wrong grading
- 10-correct mastery rule
- Simple session stats
- Minimal mobile-first visual design

## Deferred

- Audio pronunciation
- Letter tracing
- Unsure/Skip grading
- Login/accounts
- Cloud sync
- Detailed per-attempt analytics

## Verification

- App starts locally at `http://localhost:19006`
- Mobile viewport can review the first screen
- TypeScript check passes with `npm run typecheck`
